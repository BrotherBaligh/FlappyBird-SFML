# 🐦 Flappy Bird Clone (C++ & SFML)

A feature-rich, fully playable Flappy Bird clone built from scratch in C++ using the SFML library. This was my Capstone Project for my first semester of my BSc in Computer Science.

> **Note to recruiters & developers:** To strictly adhere to first-semester academic constraints, this game was built entirely **without** Object-Oriented Programming (OOP), pointers, or advanced Data Structures. It showcases strong procedural programming, complex control flow, and fundamental C++ logic!

## ✨ Key Features
* **Two Game Modes:** Choose between the classic "Dodge the Pillars" mode or a custom "Dodge the Birds" mode.
* **Customization Engine:** A built-in GUI that allows players to customize the bird's color/pattern and swap out the background environments.
* **Persistent High Scores:** Utilizes C++ File Handling (`fstream`) to save and load your all-time high score across different gaming sessions.
* **Immersive Audio:** Features continuous background music and precise sound effects that trigger every time you clear an obstacle and your score increases.
* **Custom GUI:** A polished graphical interface built natively using SFML window events and sprite rendering.
